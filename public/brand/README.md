# Energy Index — Logo Pack

Sistema logo "Ticker" per Energy Index.

## File inclusi

| File | Uso |
|---|---|
| `energy-index-primary.svg` | Logo principale per header sito, copertine, presentazioni |
| `energy-index-compact.svg` | Versione ridotta per nav bar, firma email |
| `energy-index-monogram.svg` | App icon, social avatar (100×100px) |
| `energy-index-favicon.svg` | Favicon browser (32×32px) |
| `energy-index-darkmode.svg` | Da usare su sfondi scuri (logo invertito) |
| `energy-index-mono-black.svg` | Versione monocromatica per stampa B/N |

## Palette ufficiale

- **Deep Forest** `#0A3D2E` — colore primario, sfondo del badge
- **Signal Green** `#16A34A` — accento, barra laterale, freccia
- **White** `#FFFFFF` — testo su badge scuro
- **Dark Surface** `#0F1419` — sfondo scuro consigliato per dark mode

## Tipografia

- **Ticker (EIDX)**: SF Mono / Menlo / Consolas, peso 500, letter-spacing 0.06em
- **Brand name (Energy Index)**: SF Pro / Helvetica Neue / system-ui, peso 500

## Note d'uso

- Spazio di rispetto attorno al logo: pari all'altezza della "E" di EIDX
- Sotto 32px di altezza, usare il monogram (senza freccia)
- Non modificare le proporzioni del badge
- Non cambiare i colori della palette
- Su sfondi colorati non neutri, usare la versione monocromatica

## Formati derivati consigliati

Per progetti che non supportano SVG, convertire in PNG alle seguenti risoluzioni:
- Primary: 1360×272px (4×), 680×136px (2×)
- Compact: 680×176px (4×), 340×88px (2×)
- Monogram: 1024×1024 (app icon iOS/Android), 512×512, 192×192
- Favicon: 32×32, 16×16 (PNG o ICO)
